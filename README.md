* Clone current repo  
`git clone <clone-url>` 
* Direct download link
`<link>`     
* Update Liferay Home path in build.gradle file. 
* Locate to project or import into your IDE run following gradle tasks.  

````
mvnw clean install
mvnw bundle-support:deploy

OR

mvn clean install

````
